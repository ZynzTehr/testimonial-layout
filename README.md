# Social Proof Section

A responsive social proof section built to fulfill a Frontend Mentor challenge. This project demonstrates modern HTML and CSS practices to create an optimal layout across various screen sizes.

---

## Features

- **Responsive Layout:** Automatically adjusts the layout structure to provide the optimal viewing experience depending on the device's screen size (Mobile and Desktop).
- **Modern CSS Styling:** Utilizes CSS Flexbox/Grid for precise alignment and modern styling techniques for a clean, professional aesthetic.

---

## Technologies Used

- **HTML5:** Semantic architecture for structured content.
- **CSS3:** Custom layout positioning, responsive design using media queries, and typography styling.

---

## Design System Guide

The project adheres to the following layout and color constraints provided by the design specification.

### Layout Dimensions
- **Mobile Width:** 375px
- **Desktop Width:** 1440px

### Primary Colors
- **Very Dark Magenta:** `hsl(300, 43%, 22%)` | `#502050`
- **Soft Pink:** `hsl(333, 80%, 67%)` | `#EE68A4`

### Neutral Palettes
- **Dark Grayish Magenta:** `hsl(303, 10%, 53%)` | `#937B92`
- **Light Grayish Magenta:** `hsl(300, 24%, 96%)` | `#F7F2F7`
- **White:** `hsl(0, 0%, 100%)` | `#FFFFFF`

### Typography
- **Body Copy Font Size:** 15px
- **Typeface Families:** Arial
- **Valid Font Weights:** 400, 500, 700
